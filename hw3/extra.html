Three WURFL Capabilities

brand_name
Returns the brand of the device making the request, such as Apple, Samsung, or Nokia. A developer can test this capability to show device-specific messaging, group analytics by manufacturer, or debug issues that only appear on certain brands.
form_factor
Returns the device category, such as Desktop, Tablet, Smartphone, Feature Phone, Smart-TV, or Robot. A developer can test this capability to decide whether to serve a mobile layout, a tablet-friendly layout, or a desktop experience.
resolution_width
Returns the screen width in pixels. A developer can test this capability to choose image sizes, adjust layout breakpoints, or optimize media for smaller and larger screens.
