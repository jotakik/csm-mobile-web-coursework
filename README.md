Homepage of all webpage assignments from the Mobile Web App Development course at College of San Mateo.

[Self-intro](https://jotakik.github.io/csm-mobile-web-coursework/hw1/intro.html) to practice HTML, CSS, and jQuery.

[Wildlife for Life page](https://jotakik.github.io/csm-mobile-web-coursework/hw2/index.html) incorporates Responsive Web Design techniques in refactoring a website for mobile platforms.

[ABC Toys](https://jotakik.github.io/csm-mobile-web-coursework/hw3/index.html) is a dummy online toy store that uses TailwindCSS to enable Responsive Web Design across the site.

[PWA page](https://jotakik.github.io/csm-mobile-web-coursework/hw4/index.html) explains what a Progressive Web Application is, and is itself a PWA with a service worker that caches the app for offline use.
